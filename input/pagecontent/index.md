# NMDP FHIR IG Builder Template

This is a template Implementation Guide for authoring FHIR profiles at
[NMDP (National Marrow Donor Program / Be The Match)](https://www.nmdp.org).

## Overview

This IG provides the foundation for NMDP teams to define FHIR conformance
resources using [FHIR Shorthand (FSH)](https://fshschool.org/).

## Getting Started

See the [repository README](https://github.com/nmdp-ig/fhir-ig-builder) for
development setup instructions.
